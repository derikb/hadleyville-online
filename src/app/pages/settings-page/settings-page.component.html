Settings

Data is currently stored in your browser's local storage. To use on other browsers or devices you must export then import.

Export Data

Export your data to a file for importing into another browser.

Import Data

Import your data from an exported file. If Town data is included it will update current Town. Included NPCs and Notes will either be added or overwrite existing (based on unique id).

Clear Data

Clear all saved data from local storage. THIS IS NOT REVERSIBLE.
